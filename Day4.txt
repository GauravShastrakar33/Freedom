CSS 
----

1. inline css => highest priority

<h1 style = "color: red;">Welcome to the css session</h1>

2. internal css => same p levels
3. external css => same p levels

priority depends on order =>

------------------------------------------------------------------------------------------------------------------------
case 1. internal will have more priority =>

<link rel="stylesheet" href="style.css">
<style>
    h1{
        color: blueviolet;
        text-align: center;
    }
</style>

------------------------------------------------------------------------------------------------------------------------

case 2. external will have more priority =>
    
<style>
    h1{
        color: blueviolet;
        text-align: center;
    }
</style>
<link rel="stylesheet" href="style.css">
------------------------------------------------------------------------------------------------------------------------

always use external css

------------------------------------------------------------------------------------------------------------------------

box model
----------

    border => thickness, style, color
    padding => space inside 
    margin => space outside


🔹 1. values Rule
---------------

margin: 10px 5px 20px 8px;

Order is:
Top  Right  Bottom  Left
Clockwise ⏰


🔹 2. Values Rule
------------------
padding: 10px 5px;

Means:
Top & Bottom = 10px
Left & Right = 5px


🔹 3 Values Rule
------------------
padding: 10px 5px 20px;

Means:
Top = 10px
Left & Right = 5px
Bottom = 20px
